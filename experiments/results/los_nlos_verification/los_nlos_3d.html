Use http://localhost — not file://

Browsers block Cesium Web Workers for pages opened as local files. You may see messages such as «Refused to cross-origin redirects of the top-level worker script» and «file: URLs are treated as unique security origins»: this is not your Ion token; it is the browser security model for the file: protocol.

In this file’s folder (PowerShell or terminal):

python -m http.server 8765

Then open in your browser a URL such as http://localhost:8765/cesium_odaiba.html (use your actual file name from the directory listing).
